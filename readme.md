# 2D Platformer
2D Platformer that uses LibGDX.
Designed for HTML5 and for Desktop.

## Goals & Expectations
- Perform at 60 FPS.
- Dynamic enemies that can be created easily.
- Fast-paced action.
- User-created levels.
- Cross Platform
- Server-Client multiplayer
